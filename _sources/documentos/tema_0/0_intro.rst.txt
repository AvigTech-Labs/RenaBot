Introduccion al Repositorio
===========================

Documentación y ejemplos de uso de blockly para el Robot kit RENA V1.

.. note::
   Asegúrese de revisar las versiones de los diferentes paquetes disponibles.

El HMI desarrollada tiene de fondo los siguientes paquetes:

- Blockly V11 o superior
- ESP32
- Android Studio
- KiCAD

Las librerias para instalarse se detallan en cada sección del
repositorio.
